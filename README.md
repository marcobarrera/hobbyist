# Hobbyist

## Description

Web app that allows the user to find events that cater to thier hobbies.

## Requirements

For development, you will need ruby on rails, node, and yarn installed on your environment.

## Technologies Used

<b>Built with</b>

- Rails
- HTML
- CSS
- JavaScript

## Install

    bundle install
    yarn install
    rails db:migrate

## Start

    rails s
